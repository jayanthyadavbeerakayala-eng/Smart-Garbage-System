# Smart Garbage System for Gated Communities

An IoT-based Smart Garbage System using Arduino Uno, Moisture Sensor, IR Sensor, Ultrasonic Sensors, Servo Motor, and GSM Module.

## Features
- Automatic wet/dry waste segregation
- Dual bin monitoring
- GSM SMS alerts
- Real-time monitoring
